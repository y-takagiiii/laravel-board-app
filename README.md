# 懺悔アプリ
日頃の行いを懺悔すると神からのお告げが届くアプリです。

# デモ
[![Image from Gyazo](https://i.gyazo.com/1b3935525aed5bd209d50785c536e4b1.gif)](https://gyazo.com/1b3935525aed5bd209d50785c536e4b1)
# 機能
- ログイン
- プロフィール編集
- 投稿一覧・作成・詳細・編集・更新・削除
- 検索
- お気に入り
- ページネーション(未実装)
